# 📈 초보 개미의 멘탈 관리 주식 가계부 (V1.0)

## 1. 시나리오 개요
주식을 처음 시작한 대학생 '나'는 매번 증권 앱의 복잡한 화면을 보는 것이 스트레스다. 특히 하락장에서 멘탈이 흔들려 자꾸 손절을 하게 된다. 이를 방지하기 위해, 내가 산 가격을 정확히 기록하고, 현재가를 입력하면 냉정하게 수익률을 계산해 주며, 나만의 투자 원칙을 리마인드 시켜주는 **심플한 주식 매니저 프로그램**을 기획했다.

## 2. 주요 메뉴 및 기능 (5개 이상)
1. **내 종목 등록**: 종목명, 구매 가격(평단가), 보유 수량을 입력하여 저장
2. **현재 잔고 확인**: 현재 내가 보유한 종목 리스트와 수량 출력
3. **수익률 계산기**: 현재가를 입력하면 내 평단가 대비 수익률(%)과 평가 손익 계산
4. **목표/손절가 체크**: 설정한 목표가나 손절가에 도달했는지 여부 판단 및 알림
5. **오늘의 투자 다짐**: 매매 원칙(예: "뇌동매매 금지")을 기록하거나 랜덤 투자 명언 출력

## 3. AI 파트너십 요약
- **아이디어 브레인스토밍**: 주식이라는 주제 내에서 초보자가 구현하기 쉬우면서도 실용적인 '가계부' 컨셉 도출 (Gemini 활용)
- **기획 구체화**: 2차 과제(코딩)에서 사용할 `list`, `if문` 등의 문법을 고려하여 기능을 5개로 세분화

## 4. 문제해결(Troubleshooting) 기록
- **고민**: 처음에는 실시간 주가를 자동으로 가져오는 기능을 생각했으나, 1학년 파이썬 수준에서 API 연동은 과하게 어려울 수 있다는 점을 발견함.
- **해결**: 사용자로부터 '현재가'를 직접 `input()`으로 입력받는 방식으로 변경하여, 코딩의 난이도는 낮추고 본질적인 수익 계산 기능에 집중하기로 함.
